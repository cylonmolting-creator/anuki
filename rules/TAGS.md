# Agent Tags

Tags control which rules apply to which agents via the SSOT rule generator.

| Tag | Description |
|---|---|
| `architect` | Creates/modifies agents |
| `agent-factory` | Agent production system |
| `prompt-engineer` | Writes prompts for other agents |
| `no-code` | Non-code agent — some code rules don't apply |
| `rule-keeper` | Manages rules system |
| `code-writer` | Writes code — additional code quality rules apply |
| `researcher` | Deep research with multiple sources |
| `analyst` | Data analysis |
| `designer` | Visual design |
